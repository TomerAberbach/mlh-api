configure()

Configures this module with MLH API information. All parameters should be passed in an object, and all parameters are optional. Only parameters provided will be set. The valid parameters are clientId, clientSecret, redirectUri, and accessToken. Please visit the MLH API documentation for more information: https://my.mlh.io/docs
